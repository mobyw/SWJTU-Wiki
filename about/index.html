# 关于

# 📚Wiki

SWJTU-Wiki 整合了在西南交通大学学习与生活之中的常用的信息，包括但不仅限于学习资源、推免资料、转专业资料、校内各项服务等。

# 贡献

本站使用 VuePress (opens new window) 搭建，使用了 Vdoing (opens new window) 主题，并部署在 GitHub Pages。

主项目托管在 swjtuhub/swjtu-wiki (opens new window)：

(opens new window)

如果您想要帮助本站做得更好，欢迎通过以下方式贡献：

  1. Pull request: Fork 后，进行修改并发起 Merge 请求。

  2. 邮件提交: 发送邮件到 swjtu-study@foxmail.com 。

  3. 代提交: 添加 SWJTU-Wiki QQ 群: 631209350 (opens new window)，提交 Word 或 Markdown 文件给群主整理上传。
